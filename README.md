## fire ノート

fire ノートは、消防職員が災害現場で経験したことを記事にし、共有できるサービスです。

- ターゲット
  現役の消防職員

- 解決したい課題
  個人の経験値には限りがあり、多様化する災害現場においてはこれが弱みになるため、それぞれのユーザーが経験したことを共有して、この課題を解決したい。

消防職員間で経験を共有することにより、次に自分が遭遇した災害現場で活かすことができる。また経験の少ない若手職員の育成に役立てられる。また経験したことのふりかえりができる。
